Initial Wireframe

We created an initial wireframe to communicate the new design with Engineering and Product team.
Click here to see the initial wireframe for MVPD.

Bulk Upload

In the old system, the user can only add MVPD records one at a time. In our new design, we added the Bulk Upload function to the MVPD Admin system to optimize the information entry process. After bulk uploading MVPD records, the user can even choose to make the newly uploaded batch into a whitelist group right away.

Invite an Outside User

In the new design, we allow internal user of the MVPD Admin to invite outside MVPD contact person to add records into the Admin system. The outside users are given a certain level of permission (including basic information, support information, logo, and test credentials), so that they can enter their MVPD records without access to other MVPDs.

Entitlements

We redesigned the process for brand entitlement setting up process, and significantly decreased the number of clicks for users.

Transfer and Decommission

The new workflows we designed allow users to transfer old MVPD records to a new record, and demission a brand much easier than before.
